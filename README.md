# TranCodes Portfolio

This project is to showcase Diana's Portfolio. Refactored from create-react-app to Vite.

## Technology Used

- React + Vite
- Three.js via react-three drei and react-three fiber
- TailwindCSS
- [EmailJS](https://www.emailjs.com/)
- Hosted by Google Firebase
- GitHub Actions

### Available Scripts

- npm run dev
