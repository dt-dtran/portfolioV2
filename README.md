# TranCodes Portfolio

This project is to showcase Diana's Portfolio.

## Technology Used

- React
- Three.js via react-three drei and react-three fiber
- TailwindCSS
- [EmailJS](https://www.emailjs.com/)
- Hosted by Google Firebase

### Available Scripts

- npm run dev

clear:

- rm -rf node_modules/
- rm -rf package-lock.json
- npm cache clean -f
